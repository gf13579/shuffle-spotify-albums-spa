# Shuffle Spotify Albums SPA

Literally just a big button to play a random album from my Spotify library.

## Setup

- Create a Spotify app: https://developer.spotify.com/documentation/web-api/concepts/apps
- Update the CLIENT_ID and REDIRECT_URI values in index.html
- Host index.html somewhere - matching the REDIRECT_URI
- Start using Spotify on whatever device you want to use for listening (e.g. phone --bluetooth--> speakers)

## Usage

- Browse to the site, connect to Spotify (if not already connected)
- Hit the shuffle button
